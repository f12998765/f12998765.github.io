搭建 Java 开发环境 linux
/java /总结 /linux /开发环境
11月18日 2016

安装 JDK

  1. 下载压缩包

    • 地址：http://www.oracle.com/technetwork/java/javase/downloads/index.html

  2. 解压到 /usr/local/jdk

  3. 配置环境变量

编辑

vi /etc/profile

在文件末尾添加

export JAVA_HOME=/usr/locl/jdk/**

export JRE_HOME=${JAVA_HOME}/jre

export CLASSPATH=.:${JAVA_HOME}/lib/tools.jar:${JRE_HOME}/lib/dt.jar

export PATH=${JAVA_HOME}/bin:${JAVA_HOME}/jre/bin:$PATH

# 使配置生效
source /etc/profile

安装 Tomcat

  1. 下载压缩包

    • 地址：http://tomcat.apache.org/

  2. 解压到 /usr/local/tomcat

安装 IDEA

  1. 下载安装包

    • 地址：https://www.jetbrains.com/idea/

  2. 解压到 /usr/local/

  3. 运行 /bin/idea.sh

提示错误 “Please validate either IDEA_JDK, JDK_HOME or JAVA_HOME environment variable points to valid JDK installation.”

解决：

在 */idea/bin/idea.sh 中第二行添加

export IDEA_JDK = (jdk的路径/和JAVA_HOME 相同)

相关地址： http://stackoverflow.com/questions/8382641

安装 Maven

  1. 下载安装包
    • 地址：https://maven.apache.org/
  2. 解压到 /usr/local/
  3. 更换国内源

编辑配置文件

vi /maven/conf/settings.xml

在 <mirrors> 标签(146行)中添加

<mirror>
    <id>nexus-aliyun</id>
    <mirrorOf>*</mirrorOf>ben
    <name>Nexus aliyun</name>
    <url>http://maven.aliyun.com/nexus/content/groups/public</url>
</mirror>
  1. 更多查看 http://xizero.com/notes/scraps-of-notes/

安装 DataGrip

  1. 下载安装包

    • 地址：https://www.jetbrains.com/datagrip

  2. 解压到 /usr/local/

  3. 运行 /bin/DataGrip.sh
