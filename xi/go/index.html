11月21日 2016 :cloud:

这几周一直在迷茫，明知自己没有多少东西，但还是提不起劲来。

“意识到了自己没有办法成为 top 1% 的程序员，还应该选择程序员的道路么？”

看到这句话的时候，的确很震惊，这不就是我的未来吗？

从二流的大学生成为二流的程序员……

瞬间秒杀掉了我所有的希望。

可是，没有人天生就是 NO1 ，你为什么不能成为那个 top 1%?

平地起高台，万事从头来。

你需要是一个一个的小目标，在不经意间你就会发现，原来那个遥不可及的目标已经没那么困难了。

我的小目标：

  • 总结整理 java web 基础
  • 总结整理 java web 框架
  • 学习计算机网络
  • 学习操作系统

一个年前的目标

  • 阅读 Spring 源码
