Java 继承和封装

封装

隐藏对象的属性和实现细节，仅对外提供公共访问方式

一般将成员变量的访问修饰符设置为private，通过public的getName()和setName()进行访问和赋值。

继承

类的继承是属性和方法的传递。

关键字: extends

可以用instanceof判断对象是否属于当前类或者其子类。

父类优先子类构造。

Object 是所有的父类。

this和super

this代表当前类，super代表父类，可以通过点运算符调用成员方法和成员方法。

可访问性

子类可以访问超类的公有和保护方法字段，不能访问私有方法字段。

如果，子类与超类在同一个包中，也可以访问超类的默认方法和字段。

方法的重写/覆盖

子类中的函数，函数名、参数、返回值类型与父类中某方法完全相同，则为方法的重写。

访问权限不可以缩小。

类型转换

向上强制转型 ：子类 –> 父类

Child child = new Child();
Parent parent = child;

向下强制转型： 父类 –> 子类

前提：父类的引用必须已指向了子类的一个实例

Child child = new Child();
Parent parent = child;
Child child_ = (Child) parent;
