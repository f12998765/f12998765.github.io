Collection：集合层次结构的根接口,主要的子接口是List、Set。

List：一个有序的 Collection（序列），元素可重复，常用的实现类是ArrayList、LinkedList。

Set：一个无序的 Collection（集合），元素不可重复,e1.equals(e2)=false，常用的实现类是HashSet、EnumSet。

Map：一个键值对的集合（Key - Value），映射的键不可以重复，且最多映射到一个值，常用的实现类是HashMap、EnumMap。

Iterator：遍历集合的迭代器，不可以遍历Map，ListIterator专门用于遍历List。

Arrays和Collections是操作数组、集合的工具类。
