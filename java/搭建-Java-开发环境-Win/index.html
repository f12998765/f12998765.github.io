搭建 Java 开发环境 win

安装JDK

JDK：Java Development Kit，java的开发工具包和运行环境

JRE：Java Runtime Environment，java程序的运行环境

1、打开下载页面 <下载链接>

2、选择对应系统的java 开发工具包下载 Java SE Development Kit

3、运行安装程序，默认安装

配置环境变量

打开： 控制面板 –> 系统和安全 –> 系统 –> 高级系统设置 –> 环境变量

如果使用1.5以上版本的JDK，不用设置CLASSPATH环境变量

变量名 变量值
JAVA_HOME C:\Program Files\Java\jdk1.8.0 //根据实际路径配置
Path %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;

验证

win+R –> 输入cmd –> 运行命令

java
javac
java -version

javac命令和java命令的区别 Javac：编译.java文件，生成.Class文件 Java：加载类库，运行.class文件
